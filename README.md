# AInovel

外交官の俺、戦えない魔王に転生したが交渉だけで世界を支配する

Claude Codeで直接執筆を依頼するためのリポジトリです。

## 執筆の依頼方法

Claude Codeのセッションでこのリポジトリを開き、以下のファイルを参照した上で執筆を依頼してください。

```
「settings/とknowledge_base/の内容を読んで、第X話を執筆してください。
プロット：〇〇〇〇〇〇」
```

## ファイル構成

### 設定ファイル（settings/）

| ファイル | 内容 |
|---------|------|
| `settings/world.md` | 世界観・国家・地理・魔法体系 |
| `settings/characters.md` | キャラクター設定・関係性・口調 |
| `settings/plot_overview.md` | あらすじ・今後の展開・執筆ルール |

### 執筆パターン（knowledge_base/）

| ファイル | 内容 |
|---------|------|
| `knowledge_base/novel_analysis.md` | 文体・ナラティブ構造の分析 |
| `knowledge_base/narou_patterns.md` | なろう系読者向け執筆パターン |

### 既存話数（episodes/）

第1〜5話のMarkdownファイル。文体参考・文脈維持に使用。

### その他

- `novel_config.json` — 小説基本情報（タイトル・主人公名・文字数目安など）
